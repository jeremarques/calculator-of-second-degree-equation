# Calculator of Second Degree Equation

This is a great tool for students, this calculator shows the value of the roots of the 2nd degree equation from the Bhaskara Formula, and it also calculates the value of Delta and tells if the roots are real, irrational or equal from the value of the Delta of the equation.
